Recruiter Log & Respond Workflow

n8n OpenAI API Gmail API Google Sheets

n8n automation workflow that classifies incoming emails with GPT-4, auto-labels Gmail, summarizes recruiter outreach, and logs everything to Google Sheets.

Problem

A busy inbox with recruiter outreach, promotions, and personal emails requires constant manual triage. Recruiter emails specifically need tracking — who reached out, when, and about what — but manually logging each one breaks focus and leads to missed follow-ups.

Approach

  • Gmail Trigger: n8n polls for new emails every 5 minutes
  • AI Classification: OpenAI GPT-4 mini categorizes each email into one of 7 types — Promotions, Social Media, Personal, Sales, Recruitment, Receipts, or Misc
  • Auto-Labeling: Gmail labels are automatically applied based on classification
  • Summarization: GPT-4o-mini generates a concise summary of each email
  • Logging: Classified and summarized emails are appended to a Google Sheets spreadsheet ("Email Summary n8n")
  • Routing: Conditional routing sends different email types through different processing paths

Technical Details

  • Built entirely in n8n (visual workflow automation platform) — no custom code needed
  • Text Classifier node uses GPT-4 mini with structured output for reliable categorization
  • Separate OpenAI node handles summarization with GPT-4o-mini for cost efficiency
  • Gmail OAuth2 integration for secure inbox access
  • Conditional branching handles each email category differently
  • Google Sheets acts as a lightweight CRM for recruiter tracking

Impact

  • Zero manual triage — emails are classified, labeled, and logged automatically
  • Recruiter outreach is tracked in a searchable spreadsheet for easy follow-up
  • AI summaries save time reading lengthy recruiter emails
  • Demonstrates practical LLM integration beyond chat — using AI as a classification and extraction engine in a production workflow
