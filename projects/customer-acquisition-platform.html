Customer Acquisition Analytics Platform

PySpark Spark SQL Pipeline Orchestration BI Dashboards NLP/Chat Agent

Built a production analytics platform covering 11 customer acquisition dimensions across a major e-commerce marketplace, evolving from an ad-hoc promotion analysis into a VP-endorsed enterprise solution with natural language querying.

Problem

No standardized way to measure customer acquisition across different dimensions (new-to-brand, new-to-category, new-to-marketplace, etc.). Teams relied on ad-hoc analyses that couldn't be replicated or scaled across promotion events like major sales events or seasonal campaigns.

Approach

Developed a deterministic customer classification methodology using set-difference logic, then built a parameterized Spark pipeline that processes customer order data across 11 acquisition dimensions. Designed the architecture to be event-driven — reusable for any promotion period without code changes.

Technical Details

  • EXCEPT-based set difference methodology for deterministic new-customer classification across 11 dimensions
  • Parameterized Spark pipeline (iterated through 21 versions) with configurable lookback periods
  • Unified lookback table optimization reducing redundant I/O across all dimension calculations
  • Interactive dashboard with Summary and Dimension Deep Dive tabs
  • Natural language query agent enabling self-service analytics without SQL expertise
  • Dual-level reporting: dimension-level aggregates + item-level breakdowns

Impact

  • Powers customer acquisition metrics for vendor negotiations and campaign planning across Retail, Marketing, and Partnership teams
  • Initial prototype directly convinced a top partner to invest in a second promotion iteration
  • VP of the division endorsed the platform and suggested sharing across the entire business unit
  • Customer methodology standardized across executive reporting
  • Supports a $485M+ strategic growth pillar
